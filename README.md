# QA UI Skill

A Copilot Skill for pre-launch product review. Use this skill to catch bugs, broken flows, and trust-killers before you ship.

## 🎯 What This Skill Does

When you ask Copilot to "check my app", "QA this for me", or "is this ready to ship?", this skill activates and:

- ✅ Walks through your product like a real user
- 📸 Takes screenshots of problems
- 🚨 Delivers a prioritized fix list with concrete instructions
- 🗣️ No jargon — just what's broken and how to fix it

Perfect for founders, creators, and solo builders doing a final pre-launch check.
